## How to Use

### Local Viewing
1. Open `index.html` in any web browser
2. Navigate using the sidebar or main navigation
3. Use the search box to find specific concepts

### Publishing Online
You can host this on:
- **GitHub Pages** - Free, just push to a repository
- **Netlify** - Drag and drop the wiki-site folder
- **Vercel** - Simple deployment
- Any static hosting service

### Expanding the Wiki

To add more content:

1. **Create new pages** in the `pages/` folder following the existing template
2. **Add links** in `index.html` sidebar and navigation
3. **Update search data** in `js/script.js`
